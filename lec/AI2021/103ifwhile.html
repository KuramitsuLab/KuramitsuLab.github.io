条件分岐と繰り返し¶

健康診断プログラムを作ってみましょう。BMI(ボディマス指数)は、次の式で求められる肥満度を表す体格指数です。

BMIを計算するプログラム

[2]:

h = int(input('身長(cm)'))
w = int(input('体重(kg)'))
print(w/(h/100)**2)


22.03856749311295

プログラムは、BMIを計算するプログラムのように、上から順番に評価してゆきます。

条件分岐¶

条件によっては、評価するプログラムを[X]か[Y]のように切り替えたいことがあります。 そのようなときは、if文と呼ばれる制御構造を用います。

if文や条件分岐の考え方は難しいものではありません。少し例題をときながら書いていきましょう。

例題: 食べ過ぎに注意

ユーザから身長(cm)と体重(kg)を入力し、 BMI25以上なら、「食べ過ぎに注意」と警告するプログラムを書きましょう。

ifは、分岐されるプログラムの部分をインデントすることで区別します。

[5]:

h = int(input("身長(cm)は?"))
w = int(input("体重(kg)は?"))

h = h / 100 # cm から mに変換
BMI = w / h**2

if BMI >= 25.0:
    print("食べ過ぎに注意")


食べ過ぎに注意

食べ過ぎに注意でなければ、「この調子です」と標準するようにしましょう。

``else``節: そうでなければ

if BMI >= 25.0:
    print("食べ過ぎに注意")

Python では、TrueとFalseで真偽を表現します。

数当てゲーム¶

数当てゲームとは、コンピュータが生成した０〜１００の整数をユーザが当てるまでの回数を競うゲームです。 コンピュータは、ユーザが正解を当てるまで、「大き過ぎる」「小さ過ぎる」とヒント、ユーザが入力した数値のヒントを返します。

乱数¶

乱数を生成するときは、randomモジュールを使います。

[14]:

import random
n = random.randint(0, 10)

発生した乱数を確認する

print('乱数', n)

ユーザに数を当てさせます¶

まず、ユーザに数当てをしてもらいます。 ユーザが入力した整数をmとします。

乱数nと等しければ"正解"、もし小さければ"小さ過ぎる"、そうでなければ"大き過ぎる"と表示するようにします。

[16]:

import random
n = random.randint(0, 10)

m = int(input('数字を当ててみて'))
if n == m:
    print('正解')
elif m < n:
    print('小さ過ぎる')
else:
    print('大き過ぎる')



小さ過ぎる

もちろん、一回で数を当てられるのは稀です。 ゲームにするためには、繰り返して、ユーザに数当てしてもらうようにします。

ここで、ループの構文while文かfor文を使います。

ループの構文

繰り返しのことをループ(loop)と呼びます。

while文による繰り返し¶

while文は、特殊な条件分岐の構文になります。 if文の代わりに、while文を用いると、 条件PがTrueの間はブロック``[X]``を繰り返します。

次は、while文によって、ユーザが入力した数が当たるまで繰り返す例です。

[18]:

import random
n = random.randint(0, 10)

m = int(input('数字を当ててみて'))
c = 1  # 何回で当てたか数える

while m != n : # n と mが等しくない間
    if m < n:
        print('小さ過ぎる')
    else:
        print('大き過ぎる')
    m = int(input('もう一度、数字を当ててみて'))
    c += 1  # c = c + 1

print('正解')
print(c, '回で当てました！')


正解
0 回で当てました！

ちょっと、ループの制御構造が複雑なので最初は難しいかも知れません。 強制的にループを抜け出すbreak文を使うともうわかりやすく書けます。

ループしないバージョン

[19]:

import random
n = random.randint(0, 10)

m = int(input('数字を当ててみて'))
if n == m:
    print('正解')
elif m < n:
    print('小さ過ぎる')
else:
    print('大き過ぎる')



大き過ぎる

無限ループ

while文の条件がいつまでも真で繰り返しが止まらなくなること (プログラムのバグのひとつ)

break版

[22]:

import random
n = random.randint(0, 10)

while True:  # 常に真です。
    m = int(input('数字を当ててみて'))
    if n == m:
        print('正解')
        break  # 中断
    elif m < n:
        print('小さ過ぎる')
    else:
        print('大き過ぎる')

for文: 指定された回数をループする¶

for文は、指定された回数繰り返す構文です。 プログラム中で繰り返す部分をfor文のブロックに入れます。

break文は、while文やfor文による繰り返しを強制的に中断する制御構造です。 break文を使うと、xとyが等しいとき、繰り返しを中断するようにします。

[24]:

import random
x = random.randint(0, 10)

for c in range(10):
    y = int(input('数字を当ててみて'))
    if x == y:
        print('正解')
        print(c+1, '回で当てました！')
        break
    elif y < x:
        print('小さ過ぎる')
    else:
        print('大き過ぎる')



小さ過ぎる
正解
2 回で当てました！
