1. (対話システムっぽい)相性占い¶

2021年度の倉光研究室OJT(On the Job Training, 新人研修)で 対話システムの開発が人気が高かったので、そのまま後期プログラミング演習の題材にします。

1.1. 対話システム¶

対話システムとは

人間と対話する機械またはソフトウェアをシステムのこと

対話とは、自然言語でコミュニケーションを行い情報を授受すること

対話システムとは： https://qiita.com/Hironsan/items/fd9e62c3d1ad23a67300

Conversational AI: https://sites.google.com/view/deepdial/

1.1.1. 種類¶

  • タスク指向型: 解決すべきタスクがある

  • 非タスク指向型: 解決すべきタスクがない

アキネーター API を使ってみた（サンプルコードあり） http://dotnsf.blog.jp/archives/1078447221.html

1.1.2. 構成技術¶

対話システムは、一見、単純そうにみえますが、実は色々なスキルが学習できます。

  • フレーム

1.2. 相性占い¶

対話システムっぽいけど、まず対話システムでないものを練習で作ってみましょう。

相性占い

あなたと相手の誕生日を聞いて相性を占うシステムを作ってみましょう。

要求: * (占いなので)科学的根拠はいらないが、それっぽい方がいい * 同じ入力に対して、同じ結果になる

とても簡単なコード例

[ ]:

your_birthday = input('あなたの生年月日は？')
birthday = input('相手の生年月日は？')
match = (hash(your_birthday)+hash(birthday)) % 101
print(f'相性は{match}%')

ハッシュ値を計算しておしまいというのは、 あんまりですね。もう少し占いっぽい仕上がりにしてみましょう。

1.3. ソウルナンバー占い¶

「ソウルナンバー占い」を参考にして、 相性占いをするように直してみよう。

1.3.1. ステップ1. 生年月日を書き出す¶

1987年3月23日の生年月日の人は、文字列で'19870323'とします

[ ]:

birthday = '19870323'

1.3.2. ステップ2. すべての数字を足す¶

'19870323'は、1 + 9 + 8 + 7 + 0 + 3 + 2 + 3を計算します。

Pythonで書くとこんな感じで、各桁の数字が取れます。

[ ]:

for d in birthday:
    n = int(d)
    print(n)

あとは、総和を取る部分を書いてみましょう。

1.3.3. ステップ3. 和が1桁になるまで足す¶

繰り返しを使います。

[ ]:

n = input('誕生日は？')
while len(n) > 2:
    n = str(sum(int(x) for x in n))
print('ソウルナンバーは', n)

なお、本来のソウルナンバーは、ゾロ目(11,22,33,44)は もう足さなくて良いらしいです。 正確にソウルナンバーを計算できるように、修正しておきましょう。
